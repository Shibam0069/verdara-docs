# Verdara

Verdara (VRDA) is an experimental modular blockchain designed to explore scalable decentralized infrastructure and ecosystem-driven innovation.

## Overview
Verdara focuses on building a flexible blockchain environment where developers can experiment with decentralized applications, governance models, and interoperable blockchain systems.
